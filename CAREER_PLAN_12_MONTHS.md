# 12-Month Placement Preparation Plan

**Target:** Product company offer in Data Analyst / ML Engineer / Data Scientist / SDE-AI role
**Current baseline:** Python comfortable, basic DSA, Pandas/NumPy, 2 ML projects, weak SQL & System Design
**Time budget:** 2 hrs/weekday × 5 + 8 hrs/weekend × 2 = **34 hrs/week ≈ 1,700 hrs/year**

## 12 Common Mistakes Final-Year AI/DS Students Make

1. Tutorial hell: finishing 5 courses and doing 0 projects.
2. Treating DSA as optional because "I'm AI/ML".
3. Weak SQL — a killer for analyst roles.
4. Notebook-only projects with no deployment.
5. Skipping system design entirely.
6. Starting aptitude prep in month 11.
7. Applying too late — windows close in Jan/Feb for many product companies.
8. Only targeting "ML Engineer" — Data Analyst, Data Engineer pay well too.
9. No mock interviews until the week before.
10. Chasing LangChain/LLMs before understanding what cross-entropy loss is.
11. Not leveraging seniors for referrals.
12. Perfectionism on one project instead of shipping 3 imperfect ones.
