DailyPlanner

User Guide — Daily Workflow & Best Practices

🌅 Morning Routine (5-10 mins)

1. Open Planner — your Google Calendar-style day view

  • Review today's events at a glance
  • Use Smart Planner to quickly add your day — type natural language like 9-10 Standup, 10-12 Deep work, 14-15 Review
  • Tag important events with Eisenhower quadrant (Do/Schedule) right in the event modal

2. Check Eisenhower Matrix

  • See what's tagged as 🔥 DO NOW — this is your focus list
  • Items from planner events + project tasks auto-appear here
  • Use Copy Previous Day if your routine is similar
  • Quick-add any standalone tasks for the day

3. Check Health & Habits

  • Date strip shows today — log morning habits (meditation, water, etc.)
  • Takes 30 seconds — use slider + toggle inputs
  • Completion ring updates in real-time

💼 During the Day

4. Track Time in Planner

  • Drag events to reschedule if plans change
  • Click events to mark status or edit details
  • Use 3-day or week view to see what's coming

5. Work on Projects

  • Open your active project → task table shows priorities
  • Use the side panel to log actual hours as you work
  • Check off subtasks as you complete them (visible inline under each task)
  • Tag tasks with Eisenhower quadrant when priorities shift

6. Save Links as You Browse

  • Find a useful article/video → go to Inbox
  • Paste URL → auto-fetches title, description, category via AI
  • Set status: Unread → Reading → Done → Saved

🌙 Evening Review (5 mins)

7. Daily Summary

  • See what you completed vs planned
  • Check habit completion ring and streak
  • Click AI Day Recap for insights on what went well

8. Log Remaining Habits

  • Water intake, exercise, reading pages — use +/- buttons or slider
  • Boolean habits (meditation, prayers) — toggle Done

9. Write Reflection

  • Health page → Notes field
  • Prompt: "How was today? What could improve? What am I grateful for?"

📊 Weekly Review (Sunday, 15 mins)

10. Weekly Summary

  • Review: Focus hours, completion %, habit consistency
  • Check the activity chart (total vs completed per day)
  • Click AI Weekly Recap for patterns and suggestions
  • Read your reflection highlights from the week

11. Project Health Check

  • Visit each project → check progress bar, overdue count
  • Export CSV if you need to share status with others
  • Plan next week's tasks — set due dates and priorities

12. Portfolio Check (if using)

  • Click Refresh Prices → see live P&L
  • Review XIRR trends across asset types
  • Log any buy/sell transactions

💡 Tips for Sticking With It

TipWhy
Start with planner + 3 habitsDon't try to use everything on day 1. Add modules gradually.
Use Smart Planner text inputFastest way to plan a day — type natural language, not click forms.
Tag Eisenhower quadrants on eventsForces you to think "is this really urgent?" before scheduling.
Keep projects under 30 active tasksLarger projects → break into sub-projects for clarity.
Review weekly, not just dailyPatterns only show up over a week. The AI recap helps spot them.
Use Reference Cards for SOPsAnything you do repeatedly — save the steps once, never forget.
Set cron for portfolio snapshotsIf you track investments, daily snapshots = meaningful trend charts.

🔗 The Power Combo — How Modules Connect

Project task → tagged as 🔴 Do → appears on Eisenhower page
        ↓
Scheduled in Planner V2 → shows on calendar with red "DO" badge
        ↓
Complete it → progress updates in project → habit streak counts
        ↓
Weekly summary → AI sees pattern → suggests adjustment
Key insight: Everything connects. You don't need to use all modules — pick what fits your workflow and expand from there. The Eisenhower matrix becomes most powerful when it automatically pulls from your planner and projects.
